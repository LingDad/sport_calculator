# 🏃 小学生体质健康测试计算器

基于《国家学生体质健康标准》（2024年修订），支持小学1-6年级男女生体测成绩计算。

## 测试项目

| 年级 | 测试项目 |
|------|---------|
| 1-2年级 | BMI、肺活量、50米跑、坐位体前屈、一分钟跳绳 |
| 3-4年级 | + 一分钟仰卧起坐 |
| 5-6年级 | + 50米×8往返跑 |

## 功能

- ✅ 各项成绩自动评分
- ✅ 跳绳超满分自动加分（最高+20分）
- ✅ 加权总分计算 + 等级判定
- ✅ 柱状图 + 雷达图可视化
- ✅ 适配 Streamlit Cloud 部署

## 运行

```bash
pip install -r requirements.txt
streamlit run sport_calculator.py
```
